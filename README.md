# Python AI/ML Mini-Projects

Welcome! This repository is dedicated to building **basic Python projects** integrated with **AI/ML** concepts. The focus is on mastering foundational logic and data handling through practical application.

## 🚀 Projects Overview

We are building the following core projects:

*   **Rock, Paper, Scissors:** A classic game utilizing randomized logic or basic AI patterns.
*   **Student Management System:** A CRUD application to manage student records, grades, and information.
*   **Bank Management System:** A simulation of banking operations including deposits, withdrawals, and account tracking.
*   **Multiplication Table Generator:** A simple utility to generate mathematical tables.
*   **Weather Predictor:** A basic ML project to predict weather conditions based on historical data.

## 🛠️ Tech Stack

*   **Language:** Python 3.x
*   **Libraries:** NumPy / Pandas (for data handling and analysis)
*   **Focus:** Logic building, basic data structures, and AI-assisted coding.
